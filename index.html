FUSIÓN Launcher

Centro de lanzamiento de aplicaciones de escritorio (Python + CustomTkinter)

¿Qué es?

FUSIÓN es una app de escritorio que reúne varias herramientas: Asistencia, Programas, Siga Integral, Siga Final Pro, Instituto, Ventas/Salario, utilidades CLI en versión gráfica, y más.

Cómo ejecutar en tu PC

  1. Instala Python 3.10+ desde python.org.
  2. Instala dependencias:
python -m pip install customtkinter pillow requests

Luego ejecuta:

python FUCION.PY

Descarga del proyecto

Puedes clonar el repositorio:

git clone https://github.com/Elhacker2007/dayronsde.git
cd dayronsde
python FUCION.PY

Si usas otro nombre de repositorio, ajusta la URL según corresponda.

Limitación

GitHub Pages hospeda sitios estáticos (HTML/CSS/JS). Las ventanas gráficas de Python (CustomTkinter) no se ejecutan en el navegador. Este sitio sirve para documentación y descarga; la app se corre en tu PC.
